Cho mảng a chỉ chứa các số nguyên
Xây dựng mảng b là prefix sums của mảng a cụ thể theo công thức:
b[0] = a[0]
b[1] = a[0] + a[1]
b[2] = a[0] + a[1] + a[2]
... b[n - 1] = a[0] + a[1] + ... + a[n - 1]
với n là độ dài của mảng a.
Ví dụ: Cho a = [1, 2, 3], thì kết quả prefixSums(a) = [1, 3, 6].
Mảng b được tạo ra: [1, 1 + 2, 1 + 2 + 3] = [1, 3, 6]
